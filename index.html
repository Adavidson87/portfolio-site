myFlix

An app that shows a list of movies and allows you to register an account and create a favorites list. As well as search for movies by their titles. It contains information on the directors as well as the genre. It was build with two versions, one with React and the other with Angular. The back-end is an API that I build myself.

On the backend, the app utilizes Node.js and Express and uses REST architecture. The database for it was built with MongoDB and modeled with Mongoose. It is hosted on Github pages and deployed through Heroku.

The React version is a single-page application. It uses React, Redux, and Bootstrap.

The Angular verison uses Angular Material.
